# EGDESK SMS 시스템 공식 매뉴얼

안녕하세요! 이 문서는 EGDESK SMS (평생 무료 문자 발송 시스템)의 주요 기능과 사용법을 설명하는 공식 가이드입니다. 

## 1. 대시보드
대시보드에서는 시스템의 전반적인 상태를 한눈에 확인할 수 있습니다.
- **최근 실적 요약**: 문자 발송 건수, 성공/실패 현황.
- **주요 통계 그래프**: 주간, 월간 비즈니스 지표 요약.
- **시스템 상태**: AI 연동 상태, 데이터베이스 연결 상태.

## 2. 메시지 및 문자 관리
EGDESK의 핵심 기능인 무료 문자 발송 및 관리 메뉴입니다.
- **무료 문자 발송**: 개별 고객 또는 단체 고객을 선택하여 즉시 문자를 발송할 수 있습니다. 템플릿(자주 쓰는 문구)을 활용하면 편리합니다.
- **발송내역 모니터링**: 발송된 문자의 성공, 실패(에러 사유), 예약 대기 상태를 실시간으로 추적합니다.
- **자동 발송 설정**: 특정 조건(예: 회원가입 시, 예약 확정 시, 주문 완료 시)이 충족되면 AI가 자동으로 안내 문자를 발송하도록 시나리오를 설정합니다.

## 3. 고객 및 비즈니스 관리
고객의 생애 주기를 관리하고 마케팅 효율을 높이는 기능들입니다.
- **고객 관리**: 고객의 기본 정보, 연락처, 방문 기록 등을 관리하며, VIP 고객을 분류하여 타겟 마케팅을 진행할 수 있습니다.
- **쿠폰 관리**: 새로운 할인 쿠폰이나 포인트를 생성하고, 특정 고객 그룹에게 발급합니다.
- **예약 관리**: 식당, 미용실 등 예약이 필요한 업종에서 달력 형태로 일정을 관리합니다.
- **주문내역 / 결제내역 관리**: 고객의 상품 구매 내역 및 결제(카드, 포인트 등) 상태를 확인하고 환불 처리를 지원합니다.

## 4. 시스템 설정 (AI 비서 연동)
- **AI 비서 및 이지봇 모델 설정**: 구글 AI(Gemini) API 키를 등록하여 이지봇(EasyBot)을 활성화합니다.
- API 키가 정상적으로 등록되어야 이지봇이 로컬 저장소와 데이터베이스를 분석하여 인공지능 답변을 생성할 수 있습니다.

> **💡 이지봇(EasyBot) 활용 팁**
> 이지봇에게 "최근 1주일간 가장 많이 팔린 메뉴 알려줘" 또는 "문자 발송 실패 건수가 몇 개야?"라고 물어보시면, 
> 복잡한 화면 이동 없이 즉석에서 데이터베이스를 조회하여 결과를 알려줍니다!
